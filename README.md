# Metas Financeiras

Aplicativo React Native em JavaScript de metas financeiras.

## Rodar localmente

```bash
npm install
npm run start
```

## Estrutura do projeto

```txt
src/
  app/
    _layout.jsx
    index.jsx
    target.jsx
    in-progress/[id].jsx
    transaction/[id].jsx
    styles/
  components/
    Button/
    GoalCard/
    Input/
    ProgressBar/
    ScreenHeader/
    TransactionRow/
  data/
    goals.js
  theme/
    colors.js
    fontFamily.js
  utils/
    TransactionTypes.js
    formatters.js
```
